  • tabbedbox

    This container organizes put the elements in a container. Only one component is showed at a time and can be switched pressing its correspondent tab.

    tabbedbox:
    [attributes]
    tabs: "tabName1", "tabName2", ...
    [tag] /* This corresponds with the first tab */
    [tag] /* This corresponds with the second tab */
    ...
