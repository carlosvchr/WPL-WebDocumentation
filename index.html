  • define

    Allows to create global constants.

    To access it, the identifier must be preceded by $ symbol.

    define: identifier "html code"

    Example

    define: mainColor #ff0000
    hbox:
    background-color: $mainColor

  • import

    Import javascript or css files to make it accessible.

    import: "path/file.css"
    import: "path/file.js"

  • include

    Include webpl code where it is defined.

    include: "path/file.wpla"

About info

This project aims to simplify web development as much as possible, reducing the amount of code needed as well as the time spent in the development of complete web pages. It is also intended that the programmed code be as clean and simple to understand as possible to facilitate maintenance and reduce the learning curve.
